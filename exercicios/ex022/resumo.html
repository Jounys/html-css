Usando Imagens de fundo

Como usar imagens de fundo no seu arquivo HTML5 e CSS3?

Para adicionar uma imagem como fundo no seu box-level, usa-se o comando background-image: url();.

Personalização dos fundos

Imagens que se repetem no seu site

Usando o comando background-repeat: ;, você pode ajustar formas de repetições como:

  • repeat (vai preencher a tela se repetindo)
  • no-repeat (não se repetirá na tela)
  • repeat-x (repete nahorizontal)
  • repeat-y (repere na vertical)

    • Posicionamento

    Como posicionar uma imagem no fundo

    Para isso você precisa usar o comando background-repeat e background-position: ;. Para usar o comando background-position você pode usar as definições a seguir:

  • left (esquerda)
  • right (direita)
  • top (alto/em cima)
  • bottom (em baixo)
  • center (central)

    • exemplo: left top, right bottom, center center

    Fundo Fixo

    Como deixar a imagem fixa sem quebrar?

    Com o comando background-attachment: fixed; porem quando seu site for aberto em uma tela grande, sua imagem vai quebrar, para resolver isso é preciso usar o comando background-size: cover;.
